# Discord bot

Cause of action.. <br/>
<ul>
  <li> Set up the bot suffix and token with the server (start on a test server)</li>
  <li> Set up permission based command system for the bot </li>
  <li> Feed simple commands as a basis for the system </li>
</ul>
